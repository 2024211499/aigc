集合的基本概念
集合是现代数学中最基本、最原始的概念之一，它无法用更简单的术语来精确定义，但我们可以通过描述来理解它。
定义
集合（Set）是由一些确定的、互不相同的对象组成的整体。这些对象被称为集合的元素（Element）或成员（Member）。
通常，我们用大写字母（如 A, B, C）来表示集合，用小写字母（如 a, b, x）来表示元素。
如果元素 a 是集合 A 的成员，我们称 a 属于 A，记作 a ∈ A。
如果元素 a 不是集合 A 的成员，我们称 a 不属于 A，记作 a ∉ A。
集合的表示法
列举法（Roster Method）
将集合中的所有元素一一列举出来，并用花括号 {} 括起来。
例如，由数字 1, 2, 3, 4, 5 组成的集合 A 可以表示为：
A = {1, 2, 3, 4, 5}
描述法（Set-Builder Notation）
通过描述集合中所有元素共同具有的性质 P(x) 来表示集合。
其一般形式为：{x | P(x)}，读作“所有满足性质 P 的 x 的集合”。
例如，上述集合 A 也可以用描述法表示为：
A = {x | x 是一个正整数，且 1 ≤ x ≤ 5}
几个重要的集合
在离散数学中，有几个常用的数集有特定的符号表示：
N：自然数集，通常指 {0, 1, 2, 3, ...}。
Z：整数集，{..., -2, -1, 0, 1, 2, ...}。
Q：有理数集。
R：实数集。
空集
不含任何元素的集合称为空集（Empty Set），记作 ∅ 或 {}。
子集
如果集合 A 中的每一个元素也都是集合 B 的元素，那么我们称 A 是 B 的子集（Subset），记作 A ⊆ B。
例如，{1, 2} ⊆ {1, 2, 3, 4, 5}。
任何集合都是其自身的子集，即 A ⊆ A。空集是任何集合的子集，即 ∅ ⊆ A。
